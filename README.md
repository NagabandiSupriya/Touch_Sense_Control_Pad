# Touch_Sense_Control_Pad
Touch-based control pad using STM32 for industrial applications, enabling user input through capacitive touch sensing with real-time processing. This is an embedded systems project that demonstrates a **touch-based control interface** using an STM32 microcontroller. This system enables user interaction through **touch sensing** and processes inputs in real-time, making it suitable for industrial control applications.

## 🚀 Features

- 🖐️ Touch-based input detection
- ⚡ Real-time response to user interaction
- 🔄 Efficient event handling using interrupts
- 🧠 Reliable and stable input processing
- 🏭 Suitable for industrial control panels


## 🛠️ Technologies Used

- **Microcontroller:** STM32 (e.g., STM32F429ZIT6)
- **Programming Language:** Embedded C
- **IDE:** STM32CubeIDE
- **Drivers:** HAL Drivers
- **Concepts:** Interrupts, GPIO.

## ⚙️ How It Works

1. The system detects user touch input push-buttons or keypad 
2. STM32 processes the signal using GPIO/interrupt mechanisms.
3. Based on the touch input, corresponding actions are triggered.
4. The system ensures real-time responsiveness and stability.
